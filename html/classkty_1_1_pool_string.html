Detailed Description

template<class Pool = StringPool<Sizes::stringpool_size, Sizes::string_length>, class GetPoolFunc = decltype(get_stringpool)>
class kty::PoolString< Pool, GetPoolFunc >

Thin wrapper class around a poolstring_t that redirects API calls to the string pool, as well as introducing some API shortcuts.
