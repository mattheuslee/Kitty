Detailed Description

template<int N = Sizes::stringpool_size, int S = Sizes::string_length>
class kty::StringPool< N, S >

Class that maintains all the strings in the program. The memory pool is created on the stack to avoid heap fragmentation. Holds enough memory to allocate N strings of at most length S.
